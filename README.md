# COHORT React App

This React.js project is a conversion of the original HTML page, maintaining exact visual and functional behavior.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

The app will open at `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- Exact replica of the original HTML design
- Custom cursor with smooth animations
- 3D grid floor animation
- Preloader with progress bar
- Interactive spotlight cards
- Responsive design
- All original animations and effects preserved
